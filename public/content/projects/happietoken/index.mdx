### About

HappieToken is a digital voucher platform that enables companies and individuals to buy and sell vouchers online.

I led the development team and implemented features across the full stack.

### Tech Stack

**Frontend**: Next.js, TypeScript, Shadcn UI.

**Backend**: Server Components, Kysely, PostgreSQL, ElysiaJs for public API.

**Deployment**: AWS EC2, RDS, S3.
